# Security Policy

## Scope

This repository contains personal dotfiles and system configuration for Arch Linux and related environments.

It should not contain secrets, private keys, tokens, passwords, or any other sensitive credentials.

## Reporting a Security Issue

If you find anything risky, unsafe, or secret-related in this repository, please report it privately.

Contact:

hi@elijahu.me

## What to Report

Please report:
- accidentally committed secrets
- unsafe defaults
- broken config that could expose data
- anything that looks like a security mistake

## What Not to Report Here

This repo is for configuration only. Do not use public issues to disclose secrets.

## Response

Security reports will be reviewed and handled privately.
