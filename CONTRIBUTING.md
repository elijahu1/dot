# Contributing Guide

Thanks for your interest in improving this dotfiles repository.

This repo is intentionally simple and personal, but suggestions and pull requests are welcome.

## What This Repo Is For

These dotfiles are tuned primarily for:
- Arch Linux
- Arch-based distros
- Linux desktop and server environments

Changes should keep that focus in mind.

## Contribution Rules

- Keep changes small and focused
- Do not add secrets, private keys, or machine-specific credentials
- Preserve readability over cleverness
- Keep configs practical and easy to reuse
- Avoid breaking the current setup unless the fix is clearly better

## Good Contributions

- better keybindings
- cleaner config structure
- safer defaults
- portability improvements
- bug fixes
- documentation updates

## Before You Submit

Please check that your change:
- works on a clean system
- does not depend on your personal environment
- does not add unnecessary complexity

## Pull Requests

When opening a PR, include:
- what changed
- why it changed
- any setup notes
- any side effects worth knowing
